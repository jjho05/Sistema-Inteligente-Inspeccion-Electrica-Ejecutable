# Sistema Inteligente de Inspección Eléctrica

Este es un sistema basado en Inteligencia Artificial y Agentes para la inspección eléctrica normativa.

## Características
- Análisis multimodal con Gemini Vision.
- Consulta normativa en tiempo real (RAG).
- Generación automática de dictámenes en PDF y Word.
- Interfaz interactiva y moderna.

## Despliegue
Este repositorio está configurado para ejecutarse en Hugging Face Spaces usando Docker con 16GB de RAM.
